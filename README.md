# 📄 RAG Document Assistant

A beginner-friendly **Retrieval-Augmented Generation (RAG)** application that allows users to upload a PDF and ask questions based on its content.

This project demonstrates the complete RAG pipeline using **LangChain, FAISS, and a local LLM (Ollama)**.

---

## 🚀 Features
- Upload a text-based PDF
- Ask natural language questions
- Semantic search using vector embeddings
- Grounded answers (no hallucination)
- Local & free LLM inference
- Simple Streamlit UI

---

## 🛠 Tech Stack
- Python
- LangChain
- FAISS (Vector Database)
- HuggingFace Embeddings
- Ollama (Local LLM)
- Streamlit
